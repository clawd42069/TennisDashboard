# Tennis Analyst — Dashboard v2 (Local-first)

Local-first dashboard with live updates + raw data tables + outputs.

## Run

```bash
cd tennis-analyst/dashboard_v2
source ../../tennis-analyst/venv/bin/activate  # or your venv
pip install -r requirements.txt
python -m app.server
```

Open: http://127.0.0.1:8008

## Tabs (initial)
- Matchup Report (B)
- Paper Trading Tracker (E)
- Strategies + Backtests (D)
- Player Profile (C)
